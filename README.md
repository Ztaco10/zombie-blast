# Zombie Overflow

**Description**  

Zombie Overflow is a python game that allows Players to practice their arithmetic skills while also having fun.

---

## Current Features
- Basic gameflow
- Player chooses arithmetic operation and levels
- System verifies correctness

---

## Future Work
- User gets to add custom problems to practice
- More advanced main menu/gameflow
- Zombie appears in terminal to indicate a zombie overflow
- Shop and inventory features
- Currency features

---

## Tech Stack
- **Language:** Python, JavaScript
- **Concepts:** OOP, modular design
- **Tools:** VS Code, Git/GitHub
